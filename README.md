# GeoGuessr AI

A full-stack web app that challenges you to out-guess an AI at identifying 
the location of street view images.

## What it does
- Upload a street photo and get an AI-predicted location on a map
- Play against the AI — both guess simultaneously, closest wins

## Tech stack
- Backend: FastAPI, PostgreSQL, SQLAlchemy
- Frontend: React, Vite, Tailwind, React Leaflet
- ML: PyTorch MLP on top of StreetCLIP embeddings
